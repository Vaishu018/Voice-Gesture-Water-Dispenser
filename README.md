# Smart Touchless Water Dispenser using Raspberry Pi

## Overview
This project is a smart touchless water dispenser system developed using Raspberry Pi and IoT components. It supports hot and cold water dispensing through voice commands and gesture recognition, enabling hands-free and hygienic operation.

## Features
- Hot water dispensing
- Cold water dispensing
- Voice command control
- Gesture-based touchless interaction
- Improved hygiene and accessibility
- Raspberry Pi based automation

## Technologies Used
- Raspberry Pi
- Python
- IoT
- Voice Recognition
- Gesture Recognition
- Relay Module
- IR / Gesture Sensor

## How It Works
- Users can say:
  - "Hot Water"
  - "Cold Water"
  - "Stop"
- Gesture sensor detects hand movement for touchless operation
- Relay modules control water flow
